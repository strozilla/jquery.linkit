#LinkZilla jQuery Plugin#

**LinkZilla** is a simple jQuery plugin to attach links to elements

##Version##
0.2.0

##Usage##
    $('span').linkZilla({
            href: 'https://test.com',
            text: 'Click Me',
            target: "_blank",
        });

##Vendors##
jQuery - [http://jquery.com](http://jquery.com)
